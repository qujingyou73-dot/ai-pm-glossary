# AI 产品经理术语词典

一个帮助 AI 小白打破专业词汇障碍的学习工具，双击 `index.html` 即可使用，无需安装任何环境。

🔗 **在线访问**：[点击打开](https://你的用户名.github.io/ai-pm-glossary/)

---

## 功能介绍

- **词卡学习**：点击词卡翻转，查看白话解释和生活化比喻
- **分类浏览**：6大类别，覆盖 AI 产品经理核心知识体系
- **搜索功能**：支持中英文搜索
- **随机测试**：每类10题，中英互译四选一
- **成绩排名**：测试后显示超过了多少比例的测试者
